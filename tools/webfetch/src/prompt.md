Fetches content from a URL and converts it to clean markdown.

Use this tool to retrieve and read web pages, documentation, articles, or any HTTP-accessible content.

Features:
- Automatically extracts main content from web pages (removes navigation, ads, etc.)
- Converts HTML to clean, readable markdown format
- Returns page title when available

Usage notes:
- The URL must be a valid HTTP or HTTPS URL
- HTTP URLs will work but HTTPS is preferred for security
- Large pages may be truncated to fit context limits
- Some sites may block automated requests
