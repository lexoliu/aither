Searches the web and returns relevant results.

Use this tool to find current information, documentation, news, or any web content.

Returns a list of search results, each containing:
- title: The page title
- url: The page URL
- snippet: A brief excerpt from the page

Usage notes:
- Be specific with search queries for better results
- Results are returned in relevance order
- Use web_fetch to read the full content of interesting results
